# 引言

太多人考虑的是安全而不是机会。他们似乎对生活的恐惧比对死亡的恐惧更大。

—— 詹姆斯·F·拜恩斯

这本书不会让你成为一个黑客，就像阅读军事手册不会让你成为一个士兵一样。你不会找到逐步说明如何入侵计算机的指令，也不会找到关于任何特定类型操作系统中固有的所有缺陷的技术讨论。这不是一本关于计算机黑客技术的书。这是一本关于黑客影响的哲学书。黑客不仅仅是关于入侵计算机。黑客是关于探索、扩展边界，以及为了知识本身而寻找知识。

所以，如果你在寻找关于如何在 Apache 服务器中编写 C 代码以创建缓冲区溢出的详细信息，或者你想了解如何配置 SonicWALL 防火墙以保护企业网络免受攻击，那么请另寻他处。但如果你想要一本书来探索大多数人从未见过、了解或听说过的互联网隐藏的、阴暗面的技术和社会影响，那么请继续阅读。黑客、病毒编写者、政治活动家、电话黑客、审查制度以及伪装的宣传活动都在等待着你。

不出所料，有些人会发现这本书中的信息令人反感、令人不安，甚至非常危险。有些人会将同样的信息视为造成混乱和给他人制造麻烦的借口。这两种观点都是不正确的。

这本书的目的不是教你如何成为一个黑客，而是教你如何像黑客一样思考。这意味着挑战你对对错的传统观念，并超越你文化思维方式上的限制。计算机和互联网可以打开你的心灵，让你看到你从未梦想过的新世界——或者关闭你的心灵，将你的思维引导到你选择的幻想世界的狭窄界限中。选择权在你。

如果你想要将你的计算机作为扩展你意识的一种工具，而不是作为它的替代品，那么这本书就是为你而写的。在这个世界上，我们比以往任何时候都需要你。

# 你自己的革命

请不要误解。这本书并不是在倡导推翻你的政府或发展一个截然不同的政府。相反，这本书倡导一种更个人化的革命——一种在你自己思维中的革命。而不是盲目地责怪国家政府、跨国公司、民族群体、性偏好、多元文化组织、意识形态信仰、宗教机构或政党是造成世界问题的罪魁祸首，这本书建议：

+   如果你改变你的思维方式，你也会改变你的行为方式。

+   如果你改变你的行为方式，你将能够改变他人的行为和思维方式。

+   如果你改变他人的行为和思维方式，你就可以帮助改变世界——一次改变一个人。

但一切始于你自己。

我们没有人能够 100%正确，而迈向真正变革的第一步就是承认你、我——以及你的父母、老板、配偶、家人、政府或教堂——都不可能知道一切。

不了解一切并不丢脸，但假装我们什么都懂却是一种耻辱。我们可以并且必须相互学习，无论我们的外表如何、我们住在哪里、我们信仰什么，或者我们向哪面旗帜致敬。开放、诚实的沟通是我们能够让这个世界变得更好的唯一方式，这正是本书和你的电脑发挥作用的地方。

## 沟通才是关键

尽管计算机仍然以难以使用、令人困惑和直接令人沮丧而著称，但它们在通信方面代表了一次量子飞跃，类似于字母表和印刷机的发明。有了个人电脑和互联网，人们可以发送和接收电子邮件，通过万维网研究信息，并与世界各地的人交流想法。

但不要被旨在吸引你加入计算机革命的营销炒作所欺骗。计算机和互联网的世界充满了计算机营销部门没有提到的隐藏危险，如特洛伊木马、电子间谍活动、远程计算机监控、仇恨团体、骗子、恋童癖、色情和恐怖主义——所有这些只需一点鼠标点击即可触及。

本书不仅揭示了这些危险，还帮助你理解人们最初是如何创造它们的。你对某件事了解得越多，你就越能避免或与之抗争。除了探索电视和杂志广告方便忽略的互联网地下性质之外，本书还揭示了计算机行业本身的阴暗面。

## 真相不过是观点而已

本书并不假装是关于你在互联网上可能遇到的所有可能合法或非法活动的全面资源，但它包含的信息可能会帮助或伤害他人。从根本上说，信息本身是中立的。如果你破坏了政府的计算机网络，你可能会被贴上恐怖分子的标签。如果你对敌人的计算机网络做同样的事情，你的政府可能会宣布你为英雄。善与恶完全取决于你的观点。

因此，欢迎来到计算机行业不希望你了解的那一面，一个没有光鲜印刷的教程和培训课程存在的世界。

这是真正的计算机革命的地下世界，在这里，每个人都鼓励质疑、探索和批评，但最重要的是，学会如何独立思考。

而对于许多政府、公司和宗教来说，那些知道如何独立思考的人可能是世界上最大的威胁。

# 本书内容概要

黑客活动并不局限于计算机。黑客的本质是好奇，不让障碍阻挡你的道路，它可以包括从撬锁到探索废弃建筑等各种活动。这本书中你会发现关于黑客的讨论，涵盖了广泛的话题，其中包括计算机。

书的第一部分展示了黑客技术在计算机发明之前是如何应用的。这些早期的黑客技术涉及电话系统，不出所料，这些古老的电话黑客技术在 VoIP（互联网协议语音）的世界中获得了新生，它允许人们通过互联网打电话。

电话黑客（也被称为他们更加多彩的昵称，*电话怪杰*）也开创了被称为*社会工程学*的人性化黑客艺术。社会工程学不过就是通过花言巧语诱使受害者透露有价值的信息，如密码、身份证号码，甚至是某些人的名字。尽管它起源于电话怪杰试图从不愿意透露电话系统信息的电话操作员和技术人员那里获取信息，但社会工程学以钓鱼和身份盗窃的形式重新回到了头条，黑客利用互联网对受害者进行社会工程学，窃取他们的密码、信用卡数据和社保号码。旧的东西又回来了，通过理解过去的黑客技术，你可以更好地预测未来的黑客威胁。

书的第二部分专注于早期的个人电脑黑客，他们专门破解拷贝保护软件，编写病毒和特洛伊木马，并在电子公告板上与志同道合的人交流，最终迁移到互联网。尽管他们最初有过斗争，但软件行业仍然没有找到根除这些早期黑客威胁的方法，问题反而持续增长。

书的第三部分讨论了黑客向互联网的转变，并追溯了他们的历史，从早期的拒绝服务攻击到更复杂的渗透黑客技术。你将看到互联网如何扩大了黑客的影响力和力量。

书的第四部分展示了黑客如何在一些有利益相关且支持他们的商人祝福下，为了盈利而调整了他们的技术。你将了解到企业如何操纵搜索引擎来推广他们的网站，使其在竞争对手之上，程序员如何采用广告软件和间谍软件来盈利，无论是否有人使用他们的程序，以及黑客如何用垃圾邮件淹没互联网。

书的第五部分预测了黑客未来的前景——金钱！本节讨论了所有最新的和即将到来的担忧，从广告软件和间谍软件到身份盗窃，再到横幅广告、弹出广告和搜索引擎垃圾邮件。

书的第六部分和最后一部分描述了您如何保护自己免受互联网上各种威胁，从最古老的黑客策略，包括诈骗和电话黑客技术，到最新的变化，如网络钓鱼和身份盗窃。

黑客威胁是真实的。问题是黑客带来的危险不仅仅来自恶意个人，还来自所谓的可信组织，这比任何人能编造的任何数字恐怖分子情景都要可怕。
