## 程序 14：位移编程

程序员知道左移等同于乘以 2 的幂。换句话说：

+   x << 1 等同于 x * 2 (2 = 2¹)

+   x << 2 等同于 x * 4 (4 = 2²)

+   x << 3 等同于 x * 8 (8 = 2³)

程序员使用这个技巧来快速执行简单的计算。但出了点问题：
