## 程序 72：没有尽头

这个简单的程序是将标准输入复制到标准输出。这是学生编写的第一个与输入/输出相关的程序之一。

```
  1 /************************************************
  2  * copy -- Copy stdin to stdout.                *
  3  ************************************************/
  4 #include <stdio.h>
  5
  6 int main()
  7 {
  8     // Character to copy
  9     char ch;
 10
 11     while ((ch = getchar()) != EOF)
 12     {
 13         putchar(ch);
 14     }
 15     return (0);
 16 }

```
