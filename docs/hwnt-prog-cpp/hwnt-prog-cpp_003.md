## 程序 1：Hello World

"Hello World"似乎几乎是每本编程书中的第一个程序，这个也不例外。但这个程序是出问题的。

你怎么能把像"Hello World"这样简单的东西弄坏呢？看看下面：

```
1 /************************************************
2 * The "standard" hello world program.           *
3 *************************************************/
4 #include <iostream>
5
6 void main(void)
7 {
8     std::cout << "Hello world!\n";
9 }

```

(下一个提示 228。答案 6。)

| **![开始侧边栏](img/_1.gif)** |
