# 附录 D. Ruby 和 Rails 开发软件

![无标题图片](img/httpatomoreillycomsourcenostarchimages860138.png.jpg)

要编程 Ruby，你需要一个 Ruby 解释器和编辑器或 IDE。本附录列出了 Ruby 和 Rails 开发工具的主要来源。

# IDEs 和编辑器

一些 Ruby 程序员喜欢使用简单的文本编辑器和从命令行运行程序；其他人则喜欢一个完全集成的 IDE，具有内置的调试功能。这里有一些可能性。

3rdRail

[3rdRail 官网](http://www.embarcadero.com/products/3rdrail/)

这是一个针对 Eclipse 的商用 Rails-centric IDE。它目前仅支持 Rails 1.*x*和 2.*x*。

Aptana RadRails

[Aptana RadRails 官网](http://www.aptana.com/products/radrails/)

这是一个针对 Eclipse 的免费 Rails-centric IDE。

NetBeans

[NetBeans Ruby 插件官网](http://www.netbeans.org/ruby/)

这是一个针对 NetBeans 的免费 Ruby IDE。现在，NetBeans 中 Ruby 支持的未来发展已经停止。

Ruby In Steel

[SapphireSteel 官网](http://www.sapphiresteel.com/)

这是一个针对 Visual Studio 的商用 Ruby 和 Rails IDE。

RubyMine

[JetBrains Ruby 插件官网](http://www.jetbrains.com/ruby/)

这是一个强调 Rails 的商用 Ruby IDE。

TextMate

[Macromates 官网](http://www.macromates.com/)

这是一个针对 Mac OS X 的 Ruby 编辑器。

# Ruby 实现
