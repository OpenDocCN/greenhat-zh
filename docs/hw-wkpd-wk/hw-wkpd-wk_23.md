# 附录 B. 维基百科为教师

学生使用维基百科非常普遍——这是可以理解的！维基百科易于获取，使用方便，涵盖了学生各个层次学习的大量信息。

除了阅读网站内容外，学生甚至可以做出贡献。认识到这一点，一些教师已经将编辑和分析维基百科文章纳入了他们的课程大纲。另一方面，许多教师和学校对使用维基百科有深刻的担忧。在本节中，我们将讨论在两个教育环境中使用维基百科：作为学生（在课堂内外）使用的参考资源，以及作为课堂编辑项目的可能教学材料来源。

# 维基百科作为课堂参考资源

维基百科是一个自然的参考资源：它是免费的，易于获取，并且覆盖范围比大多数参考资源都要广泛。当然，在课堂环境中使用维基百科的缺点包括可能的不准确、不完整或被破坏的内容，以及对于年幼儿童可能不适当的内容。

解决这些问题的可能方法之一是使用 CD 或 DVD 上的维基百科文章选择，如附录 A 中总结的。附录 A。其中两个选择是专门为学校设计的，一个特别针对英国国家课程。您可以在[`schools-wikipedia.org/`](http://schools-wikipedia.org/)浏览这些经过编辑和检查适合性的文章。

对于儿童或英语学习者，简单的英文维基百科可能也适用。该网站可在[`simple.wikipedia.org/`](http://simple.wikipedia.org/)找到，旨在涵盖与英文维基百科相同的话题，但以更简单的英文版本呈现。然而，该网站的广度远不及英文维基百科，而且许多文章仍然是简短的文章。

# 引导学生使用维基百科

一些学区已经禁止使用维基百科，这一举措已经成为新闻.^([37]) 当然，问题是，禁止将维基百科作为参考资料是否会真正阻止学生使用该网站——或者只是鼓励学生避免引用它。我们的看法是，学生无论如何都会使用维基百科；仅仅要求学生在作业中超越维基百科（就像任何百科全书一样）可能是一个更好的解决方案。学生应该为他们的项目引用额外的资料，而且通常好的维基百科文章可以引导学生找到这些额外资料，尽管教师应该确保这些不是学生使用的唯一参考资料。明确你对使用维基百科进行作业的期望。详细讨论维基百科及其政策也可以作为批判性地思考任何信息来源（无论是线上还是线下）的偏见和作者身份的好方法。仅仅确定一个有争议话题的中立观点可能本身就是一堂课！教学生单独评估每篇文章，使用第四章中提到的标准，也是非常重要的；维基百科不是一个单一来源，应该清楚它不是质量统一的好或差。

无论何时学生将维基百科作为参考资料，准确引用都是重要的。对维基百科的引用应始终包括（准确的）文章名称和访问该文章特定版本的链接。您可以通过在查看文章时点击左侧侧边栏上的“永久链接”或“引用此页”链接来获取这些信息。特别是“引用此页”功能提供了一个方便的方式来获取使用多种学术风格格式化的文章引用。例如，由“引用此页”功能为[[domestic sheep]]文章生成的美国心理学会（APA）风格的引用如下所示：

```
Domestic sheep. (2007, February 4). In *`Wikipedia, The Free Encyclopedia.`*
Retrieved 17:59, February 8, 2007, from http://en.wikipedia.org/w/index.
php?title=Domestic_sheep&oldid=105635704
```

# 分配维基百科编辑任务

已经开发出几个项目，将编辑维基百科文章纳入课堂活动。一些教师（主要在大学水平）甚至将贡献优秀的维基百科文章作为替代学期论文的要求。当前和正在进行中的项目列表可以在[[Wikipedia:School and university projects]]（维基百科：学校和大学项目）找到，该列表还收集了那些试图开始课堂项目的人的资源。[[Wikipedia:WikiProject Classroom coordination]]（快捷方式 WP:WPCC）也有一份愿意在任何级别上帮助教师与维基百科合作的维基百科志愿者的名单。如果你开始一个课堂项目，将这些页面上的项目列出，并让其他活跃的编辑者了解学生正在工作的文章，这可能是有帮助的；其他编辑者通常很乐意帮助回答问题并帮助学生导航维基百科。

通常情况下，学生在维基百科上写作的作业应考虑到核心政策的现实情况以及不希望的贡献将被迅速删除或更改的事实。确保学生熟悉维基百科和内容政策，并有时间尝试编辑和学习基本语法。建议让学生创建登录名（并在登录状态下进行编辑），因为这样你可以跟踪他们的贡献，如果他们与你分享用户名的话。跟踪学生的贡献也有助于你确保一切进展顺利。学生可以单独工作或作为团队，但重要的是要强调维基百科通常是一个协作项目。他们将与其他编辑者合作，这些编辑者处于学校环境之外，并且不会因为他们是学生而给予他们任何特殊考虑。

分配一篇文章进行整理或扩展可能比要求学生创建全新的文章更好。正如第五章中提到的，找到一个符合显著标准的新文章主题可能很困难；许多“低垂的果实”已经被摘取。然而，维基百科有成千上万的现有文章需要彻底重写。（如果你在需要扩展的领域与高级学生合作，要求学生开始新文章可能是个好主意。）发挥创意：如果课程专注于某个特定领域，让学生研究新信息以添加到需要扩展的文章或核实引用不当的文章。如果你在教授写作或编辑，你可以让学生练习校对或重写质量低下的文章。如果你在教授高年级学生，你可以让他们对其主题领域的文章进行评级，或者让学生在简单英语维基百科上工作，以此作为练习使用更简单的语言清晰解释信息的途径。对于任何水平的学生，为文章添加可靠的参考和来源提供了学习研究技能的理想机会。

最后，尽管可能很有诱惑力，但不要鼓励学生将专为另一个项目（如学期论文）撰写的作品直接添加到维基百科中。这类作品往往采用不适合一般百科全书的语气和风格（参见第六章）。如果你要求学生编辑维基百科文章，请将其作为与其他工作分开的独立写作项目。
