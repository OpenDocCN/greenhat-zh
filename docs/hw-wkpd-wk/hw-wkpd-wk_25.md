# 附录 D. 词汇表

维基百科的词汇表，其中包含许多在这里找不到的术语，可以在[`en.wikipedia.org/wiki/Wikipedia:Glossary`](http://en.wikipedia.org/wiki/Wikipedia:Glossary)找到。

| 管理员 维基百科的管理员；管理员是拥有额外技术权限的用户，可以删除和恢复文章、封禁账户和 IP 地址 |
| --- |
| AfD 删除文章；一个开放论坛，用于逐案讨论文章删除 |
| AGF 假设善意；一个要求编辑之间礼貌和相互信任的指南 |
| ANI 管理员通知板，用于事件，其中维基百科管理员讨论并可以提醒当前的问题，如破坏、编辑战或其他需要管理员注意的问题 |
| 匿名者 未登录时编辑的匿名或 IP 用户 |
| ArbCom 裁决委员会；一个解决现场争端的机构 |
| 裁决 ArbCom 执行的最高级别、正式且漫长的争端解决程序 |
| 回链 指向页面的维基链接，通过点击左侧边栏的“这里链接”显示 |
| BOLD 简称“大胆更新文章”，意味着如果你看到需要工作的文章，不要犹豫去改进它 |
| 机器人 在网站上运行的自动化编辑程序 |
| 官员 拥有提升其他编辑为管理员权限的权力的管理员 |
| COI 利益冲突，当文章编辑以某种方式在主题上个人投资时发生 |
| Commons *见* 维基媒体共享资源 |
| 贡献列表 编辑所做的贡献或编辑列表 |
| copyvio 简称“版权侵犯”，例如包含来自其他来源的版权材料的文章或图像 |
| CSD 快速删除标准；详细说明何时可以删除文章而不经过审查流程的政策。*另见* 快速删除 |
| 消歧义页面 一个链接到具有相似标题的替代文章的页面。例如，波士顿消歧义页面列出了至少 20 个名为波士顿的地方，以及摇滚乐队和二战时期的飞机 |
| 死胡同页面 一个没有外链的维基百科页面。*另见* 孤立文章 |
| 差异显示 显示两个维基页面版本之间差异的显示 |
| DRV 删除审查；删除文章的申诉系统 |
| 编辑 对维基页面所做的任何保存更改 |
| 编辑冲突 当两个编辑同时尝试保存页面版本时发生的冲突 |
| 编辑次数 用户账户所做的总编辑数 |
| 编辑战 两个或更多编辑之间关于文章内容的长期争端；在编辑战中，文章的更改被多次撤销，并且没有就文章内容达成共识 |
| 外部链接 指向维基百科外部的超链接 |
| 精选文章 受同行评审的维基百科文章的尊贵类别；每天在主页上突出显示不同的精选文章 |
| 精选内容 维基百科内容的一个类别，包括精选文章、图片、列表和门户 |
| 分叉（动词） 将维基分成两个编辑社区，目的是以不同的方式开发现有内容；（名词）这种分裂的最终产品 |
| GFDL 维基百科内容发布的 GNU 自由文档许可证 |
| IAR 忽略所有规则；一个传统的口号，建议维基百科用户在特殊情况下应超越形式 |
| 内部链接 在维基上指向同一维基另一个位置的超级链接；内部链接使用特殊语法而不是 URL 实现 |
| 互链 通过扩展内部链接语法来链接不同的维基；在维基百科上，链接不同语言版本的维基百科中的对应文章 |
| 引言部分 文章的介绍部分，出现在第一个标题之前 |
| 主要命名空间 主要或文章命名空间是文章出现的命名空间；此命名空间没有前缀 |
| 风格指南 维基百科写作指南的集合（WP:MOS） |
| 肉偶 由为另一派编辑谋利的人操作的账户。*另见* 木偶 |
| 调解 一种争端解决程序，调解委员会的成员试图通过正式谈判解决两个或更多编辑之间的争端，以达成共识（WP:M） |
| MediaWiki 运行维基百科网站和其他许多网站的开放源代码、免费软件 |
| Meta, Meta-Wiki 位于 [`meta.wikimedia.org/`](http://meta.wikimedia.org/) 的网站，用于协调维基媒体基金会的项目 |
| 镜像站点 发布来自维基百科（如果满足 GFDL 条件则为合法）内容的网站 |
| Monobook 维基百科的默认皮肤或模板；这是未登录或未更改用户偏好的用户看到的页面外观。*另见* 皮肤 |
| 命名空间 维基上不同类型内容的虚拟容器；命名空间由不同的前缀定义，如 *talk*：或 *Wikipedia*：，它们出现在页面名称之前；文章位于主要命名空间 |
| 重要性 文章主题应满足外部观察者的标准，通过外部来源确定 |
| 中立观点 中立观点；维基百科政策，要求文章内容以客观、中立和有根据的方式呈现 |
| 孤立文章 没有来自其他文章的 wikilinks 的条目 |
| OWN 维基百科关于文章无所有权的政策的缩写（WP:OWN）；在对话中，术语 *OWN* 可能作为不要控制文章内容的警告 |
| 页面历史  显示在“历史”标签中，按逆时间顺序显示对维基页面所做的编辑列表 |
| 永久链接 指向维基百科页面特定版本的链接 |
| 门户 收集特定主题的文章、图像和事实的项目页面 |
| PROD 提议删除；预期不会引起争议的文章删除过程 |
| 重定向（名词）仅用于将读者引导到另一个标题的文章的页面；（动词）从一个页面重定向到另一个页面 |
| 撤销 将文章恢复到先前版本；通常在编辑摘要中缩写为 *rvv* 或 *rv* |
| RfC 评论请求；争议解决程序的一部分，编辑可以讨论其他用户的行为等问题 |
| 沙盒 沙盒是留作测试编辑的页面 [[Wikipedia:Sandbox]] |
| 姐妹项目 维基百科的姐妹项目（例如，维基词典）是由维基媒体基金会设立的多语言维基集合，旨在追求一般参考相关的目标 |
| 皮肤 维基百科页面的外观；登录用户可以在用户偏好中选择维基百科页面如何显示 |
| 木偶账号 由现有账户的编辑秘密操作的第二个账户。*另见* 木偶 |
| 快速删除 由管理员进行的快速页面删除，符合快速删除政策和快速删除标准 |
| 管理员 可以更改任何用户在任何维基媒体项目上的状态的管理员 |
| 子页面 使用正斜杠（/）构建的页面，例如 [[用户:Abelard/Letter to Heloise]]；主文章命名空间不允许子页面 |
| 系统操作员 简称 *system operator*；管理员的同义词 |
| 三次撤销规则（3RR）禁止在 24 小时内撤销文章超过三次，除非是撤销破坏行为 |
| 包含 将存储在一个页面上的内容包含在另一个页面中，例如，通过向页面添加模板 |
| 转维基 将文章移动到另一个维基（通常删除原始文章） |
| 用户页面 注册用户的个人页面，通常包含有关编辑及其兴趣的信息 |
| 用户化 将文章移动到用户命名空间作为子页面 |
| 观察列表 任何登录用户可以维护的个人文章列表，列出了列表中文章的所有最近编辑 |
| 维基 一个自由可编辑的在线网页数据库，形成一个不断发展的超文本；维基百科只是维基的一个例子 |
| 维基引擎 维基的基础软件。*另见* MediaWiki |
| 维基标记语言 用于维基页面编辑和格式的特殊简化语法；也称为 *维基语法* |
| 维基化 向现有文本添加适当的 *内部链接* 并强制执行标准格式和其他房屋风格 |
| 维基律师 严格、过于字面地解释维基百科政策和习俗的人 |
| 维基链接 内部链接的同义词。*另见* 内部链接 |
| 维基共享资源 维基媒体基金会的一个项目，用于收集免费媒体（图像和视频）；维基百科文章可以包含指向 Commons 上文件的链接 |
| 维基媒体基金会（WMF）负责运营维基百科网站及其姊妹项目的非营利组织 |
| wikitext 在编辑时可见的维基页面文本（与读者所见的内容相反） |
