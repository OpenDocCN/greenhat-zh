# 附录 E. 历史

本书依赖于多年来维基媒体社区成员撰写的数千页维基百科、MediaWiki 和维基媒体基金会操作文档。根据 GFDL 许可条款，本书与维基百科一样，我们在此附录中列出我们所引用或改写的维基百科页面，以及我们所使用的维基百科图片。这些并不反映本书中的所有想法，其中许多是我们对维基百科政策和实践的解释，或本书“进一步阅读”部分中给出的所有链接。

下面的引用列出了维基百科页面的链接、所使用版本的永久链接，以及根据 User:Aka 设计的工具测量的页面前五大贡献者（不包括机器人或 IP 地址）。
